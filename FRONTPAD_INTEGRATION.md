# Интеграция лендинга с FrontPad

## Что нужно сделать

Подключить форму заказа на лендинге к FrontPad POS через REST API.
Когда клиент заполняет форму и нажимает "Заказать" — заказ автоматически создаётся в FrontPad.

## Ключ FrontPad

Секретный ключ FrontPad уже есть (используется в VK-интеграции).
Брать из проекта `integracia-vk-fp` → `callback.php` → константа `FP_SECRET`.

## Артикулы товаров

Артикулы уже заведены в FrontPad (использовались в VK Market интеграции).
Список артикулов будет добавлен сюда после уточнения.

## Как работает API FrontPad

Отправляем POST-запрос на:
```
https://app.frontpad.ru/api/index.php?new_order
```

Обязательные поля:
```
secret=...        # секретный ключ FrontPad
phone=...         # телефон клиента
name=...          # имя клиента
street=...        # адрес доставки
product[0]=...    # артикул товара
product_kol[0]=1  # количество
```

## Варианты реализации

### Вариант 1 — PHP-скрипт на хостинге (рекомендуется)
Форма на лендинге отправляет данные на `order.php` на хостинге.
`order.php` делает запрос к FrontPad API и возвращает результат.

Плюсы: секретный ключ не виден в браузере, полный контроль.

### Вариант 2 — JavaScript напрямую
Форма отправляет запрос к FrontPad API прямо из браузера через AJAX.

Минусы: секретный ключ виден в коде страницы — не рекомендуется.
