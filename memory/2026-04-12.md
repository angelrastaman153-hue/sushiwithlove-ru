### Сделано
- Лейбл оплаты: «Наличные курьеру» при доставке, «Наличные» при самовывозе (в `coSetDelivery()`)
- Разблокирована кнопка «Оформить заказ» на мобильном при открытии checkout (`coMobileBarBtn.disabled = false`)

- `_parseDelivCost` — добавлен парсинг формата «доставка - 750 руб.» для пригородов
- Зона определяется ТОЛЬКО через карту («Доставить сюда»), не через текстовые подсказки
- `coChopsticks` дефолт = 0 (был null → блокировал сабмит)
- PROMO_EXCLUDED_IDS — напитки, соусы, десерты исключены из скидки СВ10
- `getDiscountableTotal()` — учитывает isDeal, isGift, PROMO_EXCLUDED_IDS
- `coItemPriceHtml()` — зачёркнутая + новая цена на каждый товар при скидке

- Пригородный формат доставки: «доставка - 750 руб.» — добавлен второй regex в `_parseDelivCost`
